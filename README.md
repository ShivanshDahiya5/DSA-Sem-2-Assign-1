# Algorithmic Efficiency & Recursion Toolkit (AERT)

## Project Overview

This toolkit is a Python-based application developed for the **Data Structures (ETCCDS202)** course. It demonstrates the implementation of fundamental data structures, recursive algorithms, and their respective complexity analyses.

The primary goal is to compare different algorithmic paradigms—such as **Divide & Conquer**, **Dynamic Programming**, and **Greedy**—through practical coding and performance tracing.

## Key Features

* 
**Stack ADT**: A complete implementation of a Stack with `push`, `pop`, `peek`, `is_empty`, and `size` operations.


* **Recursive Analysis**: Includes recursive solutions for:
* 
**Factorial**: $n!$ calculation with base case handling.


* 
**Fibonacci**: A comparative study between a naive recursive version and a memoized version to demonstrate efficiency gains.


* 
**Tower of Hanoi**: Recursive disk movement logic with a full trace for $N=3$.


* 
**Binary Search**: A Divide & Conquer approach to searching sorted arrays.




* 
**Efficiency Tracking**: Built-in call counters to measure the number of recursive calls made during execution.
